Security policy
===============

If you believe you have found a security vulnerability in this project, please do not open a public issue with sensitive details.

Preferred reporting channels:

- Use GitHub Security Advisories for confidential disclosure: https://github.com/Shraddha0605/po-stall-agent/security/advisories
- Or open a private support request with repository owners.

Please include:

- Affected component and version
- Steps to reproduce (minimal repro preferred)
- Impact and possible mitigation

Do not include secrets (API keys, private keys, credentials) in reports or in the repository.

Response
--------
We aim to respond within 3 business days. Patches or mitigations will be coordinated via the chosen disclosure channel.
